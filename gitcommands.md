### How to git init a new project

```bash
git init

git config --local user.email <email-id>
git config --local user.name <name>

git remote add origin <git-url>
git branch -M main

git add .
git commit -m <add-message>
git push -u origin main

```

### New file changes

```bash
git add .
git commit -m <add-message>
git push
```
